## System Overview

LoanLens is a TypeScript monorepo application that processes mortgage loan documents using AI-powered extraction. The system ingests documents (PDF, DOCX, TXT, CSV, JSON, MD), parses and chunks them, then uses OpenAI's GPT-4o to extract structured borrower information with full source traceability.
